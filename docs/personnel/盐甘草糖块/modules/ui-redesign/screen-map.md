# UI 页面映射

## 共同基准

- 当前基准：`800x600`，来自 `system/Status.tjs`。
- 目标基准：`1920x1080`。
- 如果只做粗略估算，比例为 `x 2.4`、`y 1.8`。
- 推荐实现：按 16:9 原生位置重排页面并重做图片素材。纯缩放会保留旧 4:3 密度，浪费横向空间。

## 标题界面

- 负责文件：`system/Title.tjs`
- 当前资源：`FRM_0511` 标题背景 `800x600`，`FRM_0513` Logo/提示叠层，`FRM_0551`-`FRM_0556` 主菜单按钮，`FRM_0561`-`FRM_0563` Bonus 菜单按钮，`FRM_0501`-`FRM_0503` Logo/注意事项序列。
- 当前布局：完整 `800x600` 标题图；菜单按钮 `224x25`，集中在低高度按钮条中；自动存档悬停预览复用 `SavedataHeader`。
- 当前问题：按钮高度对 1080p 太小；菜单组合过时；4:3 背景无法无黑边填满 16:9；悬停状态依赖小位图条。
- 目标布局：重做 `1920x1080` 标题背景；主菜单放在右侧竖排或右下列，例如 `x=1320..1780`、`y=520..900`；底部只保留版本/版权安全条。
- 图片需求：标题背景/叠层、正常/焦点/禁用按钮图集、Bonus 子菜单按钮、自动存档预览卡片样式。

## 对话框

- 负责文件：`system/MessageFrame.tjs`、`system/MessageArea.tjs`
- 当前资源：`FRM_0101`、`FRM_0102` 对话框 `750x170`；`FRM_0105`、`FRM_0106` 等待/光标标记；`FRM_0108` 关闭按钮；`FRM_0109` 语音按钮；`FRM_0161`-`FRM_0168` 路线/名字标记。
- 当前布局：底部对话框，名字区域 `800x60`，正文区域约 `700px` 宽，系统菜单从底边滑出。
- 当前问题：750px 框在 16:9 下太窄；对话框遮挡底部画面但没有利用横向空间；文本边距按 600p 调整；菜单和语音控件拥挤。
- 目标布局：底部居中面板 `1560x210`，位置 `x=180`、`y=810`；名字牌 `360x52`；文本区域 `1360x130`；控件靠面板右侧。保留画面上方约 70% 不被遮挡。
- 图片需求：半透明对话面板、名字牌、下一步提示、语音/回放状态、关闭/隐藏控件、路线/名字标记。

## 快捷菜单

- 负责文件：`system/MessageFrame.tjs`，共享控件在 `system/SelectItem.tjs`
- 当前资源：`FRM_0122`-`FRM_0129` 快捷按钮，`FRM_0130`-`FRM_0133` 跳转子菜单，`FRM_0141`/`FRM_0142` 快速存档子菜单，`FRM_0181`-`FRM_0183` 音频/设置子菜单。
- 当前布局：`800x33` 系统菜单条，按钮图约 `164x25`，子菜单从菜单条弹出。
- 当前问题：点击目标太小；很多操作依赖文字型位图按钮；子菜单容易覆盖对话区域；坐标基于 4:3 底部。
- 目标布局：紧凑工具条 `x=1120..1800`、`y=760..812`，或隐藏对话框时放到右上；子菜单卡片向上展开并保持在 `x=1180..1800` 内。
- 图片需求：存档、读档、历史、设置、自动、跳过、跳转、标题、隐藏等图标/文字按钮图集，子菜单面板、滑条轨道/滑块、快速存档小卡片。

## 选项界面

- 负责文件：共享选择控件在 `system/SelectItem.tjs`；具体场景调用点不在本交接的修改范围内，但视觉基础控件需要支持新选项样式。
- 当前资源：主要复用按钮、单选、切换、链式导航基础控件。
- 当前布局：在 `800x600` 舞台居中。
- 当前问题：旧按钮尺寸和键盘焦点提示不适合 1080p；长文本容易挤压位图按钮。
- 目标布局：选项竖排居中，围绕 `x=960`、`y=420..760`；按钮最大宽度 `920`；项目高度 `72`；间距 `18`；焦点/选中状态要明显但不闪烁。
- 图片需求：可扩展选项按钮状态、焦点环、禁用/锁定状态、可选小箭头或标记。

## 设置窗口

- 负责文件：`system/ConfigWindow.tjs`
- 当前资源：页面底图 `FRM_02A01`、`FRM_02A02_`、`FRM_02A03`、`FRM_02A04`；标签 `FRM_02A11`-`FRM_02A14`；全局按钮 `FRM_02A21`-`FRM_02A26`、`FRM_1101`；切换/单选/滑条 `FRM_02A31`、`FRM_02A32`、`FRM_02A41`、`FRM_02A42`、`FRM_02B01`-`FRM_02B07`、`FRM_02C11`。
- 当前布局：居中底板 `640x568`，每页控件使用小固定坐标，示例消息条为 `FRM_02B35`。
- 当前问题：面板密度过高且风格旧；标签严重依赖位图；画面尺寸设置只有 4:3 缩放模式；滑条和复选框点击区小。
- 目标布局：居中界面 `1440x860`，`x=240`、`y=110`；左侧标签栏 `260px`；右侧内容 `1080px`；底部操作行。使用分组标题和行式设置项。
- 图片需求：设置外壳、标签状态、开关状态、单选状态、滑条轨道/滑块/填充、示例消息预览面板、关闭/返回/默认/初始化/标题/结束按钮。
